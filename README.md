# Vinay Trivedi - Portfolio

A professional portfolio website built with React frontend and Node.js backend, showcasing cybersecurity expertise and experience.

## Architecture

- **Backend**: Node.js + Express API serving portfolio data
- **Frontend**: React with component-based architecture
- **Design**: Hackfolio-inspired clean, professional UI
- **Content**: Strictly derived from LinkedIn profile data

## Setup Instructions

### Backend Setup
```bash
cd backend
npm install
npm run dev
```
Server runs on http://localhost:5000

### Frontend Setup
```bash
cd frontend
npm install
npm start
```
Frontend runs on http://localhost:3000
